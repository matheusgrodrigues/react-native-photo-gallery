# Photo Gallery

Aplicativo simples para demonstrar a busca e renderização de uma lista de imagens utilizando `React Native`;

-  [Picsum Photos](https://picsum.photos/)
-  FlatList
-  Image
-  Fetch API
-  TypeScript
